# PopCon Promo Flyer

This repository contains the standalone promotional site (Beta Tester Guide) for **PopCon**.

## Project Details
- Built with plain HTML/CSS.
- Images are embedded as base64 for portability.
- Favicon is located in `beta-assets/`.

## Deployment
This project is designed to be deployed to **Vercel** as a static site.
- **Vercel Name**: `popcon-promo`
- **Main Entry**: `index.html`
